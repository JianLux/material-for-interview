# 清除浮动

标签（空格分隔）： css

---

## Why?

一般来说清除浮动有以下原因：

1. 不希望某一元素两边有浮动元素
2. 浮动元素排版超出了其父级元素，父元素的高度出现了坍塌

解决第一个问题，需要清除该元素周围的浮动；
解决第二个问题，将父级元素的高度撑起来

## 清除浮动的方法

### 1. 利用clear样式
给需要清除浮动的元素添加如下样式：
```
{
    clear: left
}
```
### 2. 父元素结束标签之前插入清除浮动的块级元素
```
<div class="topDiv">
    <div class="textDiv">...</div>
    <div class="floatDiv">float left</div>
    <div class="blankDiv"></div>  // 此为清除浮动标签
</div>
<div class="bottomDiv">...</div>
```
```
.blankDiv {
    clear: both; // or left
}
```
### 3. 利用伪元素（clearfix）
```
<div class="topDiv clearfix">
    <div class="textDiv">...</div>
    <div class="floatDiv">float left</div>
</div>
<div class="bottomDiv">...</div>
```
```
.clearfix:after {
    content: '';
    height: 0;
    display: block;
    clear: both;
}
```
### 4. 利用overflow清除浮动
```
<div class="topDiv">
    <div class="floatDiv">float left</div>
    <div class="textDiv">...</div>
</div>
<div class="bottomDiv">...</div>
```
```
.topDiv {
    width: 500px;
    padding: 4px;
    border: 2px solid black;

    // 区别在这里
    overflow: auto;
}
```
